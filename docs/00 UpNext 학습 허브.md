# UpNext 학습 허브

> UpNext 앱을 만들면서 배우는 개발 개념들을 정리한 공간입니다.
> 이 폴더를 Obsidian Vault로 열면 그래프뷰로 개념 간 연결을 시각적으로 볼 수 있어요.

## 기술 스택 개요

```
UpNext
├── [[Next.js]] ─── React 기반 웹 프레임워크
├── [[TypeScript]] ─── 타입이 있는 JavaScript
├── [[Tailwind CSS]] ─── 유틸리티 기반 CSS 프레임워크
├── [[Zustand]] ─── 상태 관리 라이브러리
├── [[Framer Motion]] ─── 애니메이션 라이브러리
└── [[LocalStorage]] ─── 브라우저 내장 저장소
```

## 개발 개념 지도

```
[[웹 개발 기초 용어]] ← 프레임워크, 렌더링, SEO 등 기초 개념
    ↓
[[React 컴포넌트]] ← 화면을 구성하는 블록
    ↓ 사용
[[Props]] ← 컴포넌트에 전달되는 데이터
    ↓ 관리
[[상태 관리(State)]] ← 변하는 데이터
    ↓ 저장
[[LocalStorage]] ← 브라우저에 영구 저장
```
